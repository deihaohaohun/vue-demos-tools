# Vue3 Demo Codes

Vite, Vue3 练手项目

## todo list

### 已实现的功能

- 模板/任务/目标：创建模板与目标，并按周期自动生成当日任务
- 任务列表：未开始/已打卡分组展示；目标页签（未完成/已完成/已放弃）与数量显示
- 打卡：支持打卡备注；打卡弹窗可修改当次分钟（±1，Shift+点击为±5）
- 打卡记录：按日期查看（前一天/后一天）；支持编辑备注与当次分钟
- 数据统计：今日统计卡片、分类统计、趋势图（打卡次数/分钟）
- 连续打卡：展示连续打卡天数与最大连续天数
- 热力图：最近一年活跃热力图（打卡次数/分钟）
- 导出：按周预览与切换（上一周/下一周），支持深色模式，导出为图片（通过 snapdom 实现）
- 配置管理：分类/最小频率/每次分钟配置可维护（增删/恢复默认）
- 归档：任务可归档并在归档列表查看（归档后不再生成新任务）
- 数据持久化：使用 localStorage 保存任务/模板/统计/打卡记录等
- 页面响应式与深色模式
